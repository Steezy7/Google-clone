# Google Clone 

A responsive Google search page clone built with **Next.js** and **Tailwind CSS**, recreating the look and feel of the Google homepage. This project demonstrates UI design, layout skills, and integration with search functionality, giving users a familiar and interactive search experience.

##  Features

-  Homepage UI similar to Google  
-  Search input with redirection to search results  
-  Responsive layout with Tailwind CSS  
-  Built with modern Next.js features

##  Tech Stack

This project uses:

- **Next.js** – React framework for server‑side rendering and routing  
- **Tailwind CSS** – Utility‑first CSS framework for styling  
- **JavaScript** – Core scripting language
